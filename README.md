# SDC Chettiyankinar Website

This repository contains the source code for a sample website of the Skill Development Centre (SDC) in Chettiyankinar. The project serves as a digital prototype to showcase courses, institutional announcements, and student resources.


## Features

*   **Responsive User Interface:** Optimized for viewing across desktop, tablet, and mobile devices.
*   **Course Catalog:** Details regarding professional skill development programs.
*   **Announcements Portal:** Mock updates concerning admissions and institutional events.
*   **Contact Services:** Integrated mapping placeholder and contact forms.


## Tech Stack

*   **Frontend:** HTML5, CSS3, JavaScript
*   **Hosting:** GitHub Pages


## Local Setup

To view the project locally, clone the repository and open the main entry file:
